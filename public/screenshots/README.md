# Screenshot Guidelines

This directory contains visual assets illustrating the user flows, dashboards, and accessibility modules of EcoWise AI.

---

## Required Screenshots

To present the project professionally in competition submissions and repository headers, place 10 high-resolution screenshots here using the exact filenames below:

1. `1_landing_page.png` - Landing page with hero banner and highlights.
2. `2_onboarding.png` - Questionnaire wizard interface.
3. `3_dashboard.png` - Main user dashboard (Score, Budget gauge, Quick Actions).
4. `4_ecotwin.png` - EcoTwin prediction projection panel.
5. `5_carbon_simulator.png` - Simulator sliders and twin charts.
6. `6_sustainability_coach.png` - Coach chat console and emission hotspot analysis.
7. `7_challenges.png` - Habit-based dynamic challenges (Daily/Weekly/Monthly).
8. `8_analytics.png` - Progress graphs, breakdown donut, and Decision Intelligence diagram.
9. `9_achievements.png` - Badge grid, current level details, and historical logs.
10. `10_settings.png` - Layout preferences (Contrast, units, and data resets).

---

## Capture Recommendations

* **Window Dimensions**: Capture at a standard layout size of `1280x800` or `1920x1080` (landscape).
* **Color Schemes**: Use the default dark-mode glassmorphic interface, or capture a few in High Contrast mode to illustrate accessibility features.
* **Compression**: Run screenshots through a compression tool (e.g., TinyPNG) to ensure the total repository size remains well under **10 MB**.
